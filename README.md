# dsci551-project

Commands to run the project:
# Use venv if exists

```bash
source venv/bin/activate
```

# Initilaize DBs and import data

```bash
   python3 manage.py makemigrations
  ```
```bash
  python3 manage.py migrate
  ```
```bash
  python3 manage.py parser2
```
# To run the app

```bash
python3 manage.py runserver
```
